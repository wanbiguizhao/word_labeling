## 一、问题背景

### 1.1 动机

在字符分割任务中，人工标注成本高、效率低。通过主动学习策略，可以**智能选择最有价值的样本**进行标注，从而：
- 降低标注成本（只标注关键样本）
- 加速模型收敛（标注最有信息量的样本）
- 提升模型精度（聚焦难分样本）

### 1.2 核心假设

| 假设 | 说明 |
|------|------|
| 模型不确定的样本 | 模型预测概率接近0.5，说明模型对该样本缺乏信心 |
| 规则与模型分歧大的样本 | 规则和模型分割结果差异大，说明该样本存在歧义 |
| 难分样本包含更多信息 | 难分样本能帮助模型学习更复杂的模式 |

## 二、主动学习评分体系

### 2.1 综合评分公式

$$
AL_{score} = w_1 \cdot U + w_2 \cdot E + w_3 \cdot D + w_4 \cdot C + w_5 \cdot B
$$

| 分量 | 符号 | 权重 | 含义 | 重要性 |
|------|------|------|------|--------|
| 不确定性 | $U$ | 0.2 | 模型预测概率接近0.5的程度 | 中 |
| 熵 | $E$ | 0.2 | 预测概率分布的混乱程度 | 中 |
| **分歧** | **$D$** | **0.3** | **规则和模型分割的IoU差异** | **高** |
| 数量差异 | $C$ | 0.15 | 规则和模型识别的字符数量差异 | 低 |
| 边界位移 | $B$ | 0.15 | 规则和模型边界位置的平均差异 | 低 |

### 2.2 评分分量详解

#### 2.2.1 不确定性分数 $U$

**定义**：预测概率越接近0.5，不确定性越高

**公式**：
$$
U = \frac{1}{W} \sum_{i=1}^{W} 2 \cdot \min(p_i, 1-p_i)
$$

**解读**：
- $p_i$：第i列属于字符的预测概率（0-1）
- $\min(p_i, 1-p_i)$：距离0.5的距离（0-0.5）
- 乘以2归一化到[0, 1]范围
- **越高表示模型越不确定**

**示例**：
- $p_i = 0.5$ → $\min(0.5, 0.5) \times 2 = 1.0$（最大不确定）
- $p_i = 1.0$ → $\min(1.0, 0.0) \times 2 = 0.0$（确定是字符）
- $p_i = 0.0$ → $\min(0.0, 1.0) \times 2 = 0.0$（确定不是字符）

#### 2.2.2 熵分数 $E$

**定义**：预测概率分布越均匀，熵越大

**公式**：
$$
E = -\frac{1}{W} \sum_{i=1}^{W} \left( p_i \cdot \log(p_i) + (1-p_i) \cdot \log(1-p_i) \right)
$$

**解读**：
- 熵衡量概率分布的混乱程度
- 当 $p_i = 0.5$ 时，熵最大（$\log(2) \approx 0.693$）
- 当 $p_i = 0$ 或 $1$ 时，熵最小（0）
- **越高表示模型越不确定**

**归一化**：
$$
E_{norm} = \frac{E}{\log(2)}
$$

#### 2.2.3 分歧分数 $D$

**定义**：规则分割和模型分割结果的IoU差异

**公式**：
$$
D = 1 - \text{IoU}(\text{rule\_mask}, \text{model\_mask})
$$

**计算步骤**：
1. 根据规则分割结果生成二值掩码 $\text{rule\_mask}$
2. 根据模型分割结果生成二值掩码 $\text{model\_mask}$
3. 计算两个掩码的IoU
4. 用1减去IoU得到分歧分数

**解读**：
- IoU = 1 → D = 0：规则和模型完全一致（无分歧）
- IoU = 0 → D = 1：规则和模型完全不同（最大分歧）
- **越高表示分歧越大**

#### 2.2.4 数量差异分数 $C$

**定义**：规则和模型识别的字符数量差异

**公式**：
$$
C = \frac{|\text{rule\_count} - \text{model\_count}|}{\max(\text{rule\_count}, \text{model\_count}, 1)}
$$

**解读**：
- 当规则识别5个字符，模型识别3个字符时：$C = |5-3| / 5 = 0.4$
- 当规则和模型识别数量相同时：$C = 0$
- **越高表示数量差异越大**

#### 2.2.5 边界位移分数 $B$

**定义**：规则和模型边界位置的平均差异

**公式**：
$$
B = \min\left( \frac{\frac{1}{2N} \sum_{i=1}^{N} (|r_{start,i} - m_{start,i}| + |r_{end,i} - m_{end,i}|)}{\text{avg\_width}}, 1 \right)
$$

**计算步骤**：
1. 对齐规则和模型的字符区间（按位置匹配）
2. 计算对应区间的起始和结束位置差异
3. 除以平均字符宽度归一化
4. 取最小值不超过1

**解读**：
- $B = 0$：边界完全一致
- $B = 1$：边界差异达到平均字符宽度
- **越高表示边界差异越大**

## 三、优先级判定规则

### 3.1 优先级划分

| AL分数范围 | 优先级 | 颜色标记 | 说明 | 建议操作 |
|-----------|--------|----------|------|----------|
| $> 0.7$ | **极高** | 🔴 红色 | 模型极不确定且与规则严重分歧 | **必须标注** |
| $0.5 - 0.7$ | **高** | 🟠 橙色 | 有明显不确定性或分歧 | **强烈建议标注** |
| $0.3 - 0.5$ | **中** | 🟡 黄色 | 中等难度，存在一定歧义 | **选择性标注** |
| $< 0.3$ | **低** | 🟢 绿色 | 模型确定且与规则一致 | **无需标注** |

### 3.2 典型场景分析

| 场景 | 不确定性 | 分歧 | 数量差异 | AL分数 | 优先级 | 原因 |
|------|---------|------|---------|--------|--------|------|
| 正常印刷体 | 低 | 低 | 低 | < 0.3 | 低 | 规则和模型都能正确分割 |
| 粘连字符 | 高 | 高 | 高 | > 0.7 | 极高 | 规则可能切分错误，模型也不确定 |
| 模糊字符 | 高 | 中 | 低 | 0.5-0.7 | 高 | 模型不确定，但规则可能正确 |
| 特殊符号 | 中 | 高 | 中 | 0.5-0.7 | 高 | 规则和模型对符号分割有分歧 |

## 五、算法实现

### 5.1 评分计算伪代码

```python
def compute_al_score(line_id):
    # 1. 获取模型预测结果
    model_intervals, pred_prob = model.predict(line_id)
    
    # 2. 获取规则分割结果
    rule_intervals = load_rule_result(line_id)
    
    # 3. 计算各分量分数
    uncertainty = mean(min(p, 1-p) * 2 for p in pred_prob)
    entropy = mean(-p*log(p) - (1-p)*log(1-p) for p in pred_prob) / log(2)
    disagreement = 1 - iou(rule_mask, model_mask)
    count_diff = abs(len(rule_intervals) - len(model_intervals)) / max(...)
    boundary_diff = mean_boundary_displacement(rule_intervals, model_intervals)
    
    # 4. 计算综合分数
    al_score = (0.2 * uncertainty + 0.2 * entropy + 
                0.3 * disagreement + 0.15 * count_diff + 
                0.15 * boundary_diff)
    
    return al_score
```

### 5.2 IoU计算伪代码

```python
def iou(rule_mask, model_mask):
    intersection = sum(r & m for r, m in zip(rule_mask, model_mask))
    union = sum(r | m for r, m in zip(rule_mask, model_mask))
    return intersection / union if union > 0 else 0
```

### 5.3 边界位移计算伪代码

```python
def mean_boundary_displacement(rule_intervals, model_intervals):
    if len(rule_intervals) != len(model_intervals):
        return 1.0
    
    total_displacement = 0
    total_width = 0
    
    for (r_start, r_end), (m_start, m_end) in zip(rule_intervals, model_intervals):
        total_displacement += abs(r_start - m_start) + abs(r_end - m_end)
        total_width += (r_end - r_start) + (m_end - m_start)
    
    avg_width = total_width / (2 * len(rule_intervals))
    avg_displacement = total_displacement / (2 * len(rule_intervals))
    
    return min(avg_displacement / avg_width, 1.0)
```

| 策略 | 说明 | 适用场景 |

| **多样性采样** | 在高AL分数样本中选择多样性样本 | 避免重复标注相似样本 |

### 7.2 自适应权重

根据模型训练阶段动态调整权重：

| 阶段 | $w_1$ | $w_2$ | $w_3$ | $w_4$ | $w_5$ | 策略 |
|------|-------|-------|-------|-------|-------|------|
| 初期 | 0.3 | 0.3 | 0.2 | 0.1 | 0.1 | 侧重模型不确定性 |
| 中期 | 0.2 | 0.2 | 0.3 | 0.15 | 0.15 | 平衡不确定性和分歧 |
| 后期 | 0.1 | 0.1 | 0.5 | 0.15 | 0.15 | 侧重规则与模型分歧 |

## 九、注意事项

### 9.1 权重调整

- 权重需要根据实际数据分布调整
- 建议通过交叉验证确定最优权重
- 不同阶段可使用不同权重配置

### 9.2 计算效率

- 对4609个行图像计算AL分数约需10-20分钟
- 可通过批量推理优化速度
- 建议离线计算，定时更新排名

### 9.3 标注质量

- 主动学习依赖高质量标注
- 建议对标注结果进行抽检
- 标注标准应保持一致

### 9.4 模型更新

- 每次标注后应重新训练模型
- 建议保留历史模型版本
- 对比不同迭代阶段的模型性能
